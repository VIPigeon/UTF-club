CPU Support: Modern CPUs provide specific instructions and mechanisms to assist
with context switching. For example:

Interrupts: When an interrupt occurs (e.g., a timer interrupt), the CPU
	automatically saves the current execution context (registers, program counter)
	onto the stack or a designated area in memory.

Special Instructions: Some CPUs have special instructions that can facilitate
	context switching, such as those that can quickly save and restore
	registers.

Memory Management: CPUs support virtual memory, which allows the OS to manage
	process memory spaces effectively, making it easier to switch between
	processes.
