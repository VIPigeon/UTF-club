* Лекция 2024-02-14 [1]

# Введение

На МОЗИ мы будем заниматься алгебраическими структурами: конечные группами,
кольцами, полями. Например:
- Zn — множество классов вычетов по n
- Z^*_n - группа Эйлера

Лемма (о делении с остатком)
        ∀a ∈ Z, b ∈ N,  ∃q ∈ Z, r ∈ Z:

                a = bq + r,  0 <= r < b

Доказательство
        Пусть M = { a - bx | x ∈ Z, a - bx >= 0 }

        M содержит хотя бы одно число:
                Если  a >= 0  то  a - b * 0 = a >= 0
                Если  a < 0   то  a * (1 - b) >= 0

        Пусть r — наименьшее число из M

        Докажем, что r < b (от противного):
                Пусть r >= b, тогда r1 = r - b >= 0
                r1 = (a - bq) - b = a - b(q + 1)

                То есть r1 ∈ M и r1 < r

                Это противоречит тому, что r — наименьшее.

        Также докажем, что такая пара q, r единственна (от противного):
                Пусть             a = bq1 + r1, 0 <= r1 < b
                С другой стороны, a = bq + r

                Вычтем одно из другого: 0 = b(q1 - q) + (r1 - r)
                        =>              (r1 - r) = b(q1 - q)

                Пусть q1 != q, тогда |q1 - q| >= 1

                Рассмотрим

                        b |q1 - q| = |r1 - r|
                            >= b        < b

                |r1 - r| < b так как 0 <= r < b  и  0 <= r1 < b

                Получили противоречие.

        Лемма доказана
                
Обозначение
        Число r из леммы обозначают a mod b

# Свойства остатков

Свойство 1
        Равносильны:
        1. a mod m = b mod m 
        2. ∃t ∈ Z: a - b = mt

        Другими словами, m | (a - b)

Свойство 2
        (a + b) mod m = ((a mod m) + (b mod m)) mod m

Свойство 3
        (ab) mod m = ((a mod m) * (b mod m)) mod m




# Отношение делимости

Определение
        Число b != 0 делит число a, если

                ∃c ∈ Z: a = bc

        В таком случае пишут b | a


# Свойства отношения делимости

Свойство 1
        b | a, c | b  =>  c | a 

Свойство 2
        b | a1, b | a2, ... b | ak (k >= 2)
                =>
        ∀c1, c2, ... ck ∈ Z  b | (a1c1 + a2c2 + ... + akck)

Свойство 3
        b | a, a != 0  =>  |b| <= |a|

# Критерии "подгрупповости"
        G — группа,
        H \subset G

        Когда H подгруппа?

Теорема 1
        Пусть G — группа, H \subset G непустое.
        Равносильны следующие утверждения:

                1) H — подгруппа G 
                2) ∀a,b ∈ H  ab ∈ H и a^{-1} ∈ H

Теорема 2
        Пусть G — группа, H \subset G непустое.
        Равносильны следующие утверждения:

                1) H — подгруппа G 
                2) ∀a,b ∈ H  ab^{-1} ∈ H

Теорема 3
        Пусть H конечное подмножество группы G
        Расвносильны:
                1) H — подгруппа G
                2) ∀a,b ∈ H  ab ∈ H

# Порядок элемента группы

Определение
        Пусть G — группа.
        Элемент a ∈ G имеет конечный порядок, если

                ∃n ∈ N:  a^n = e

        Если a имеет конечный порядок, то min{n ∈ N | a^n = e} называется порядком a
        Обозначается |a|, ord(a), O(a). Если a^n != e, ∀n ∈ N, то a имеет бесконечнный
        порядок.

Свойство 1
        ord(e) = 1

Свойство 2 (условие конечности порядка)
        Равносильны:

        (1) ord(a) < ∞
        (2) ∃k,m ∈ Z: k != m  и  a^k = a^m 
        (3) ∃s ∈ N: a^-1 = a^s

Свойство 3
        ord(G) < ∞  =>  ord(a) < ∞, ∀a ∈ G

Свойство 4 (условие бесконечности порядка)
        Равносильны:

        (1) ord(a) = ∞
        (2) ∀k,m ∈ Z: k != m  a^k != a^m

Свойство 5
        Пусть |a| = n ∈ N. Тогда элементы e, a, a^2, ..., a^{n-1} попарно различные

Свойство 6
        Пусть |a| = n ∈ N. Тогда k ∈ z:

                a^k = e  <=>  n | k

Доказательство
        Пусть a^k = e. Разделим k на n:

                k = nq + r, 0 <= r <= n - 1

                a^{nq + r} = e  =>  (a^n)^q * a^r = e
                                      = e

        Тогда a^r = e  => r = 0  => n | k

        Тогда всё ясно

Свойство 7

        Пусть ord(a) = n  ∀k ∈ Z

        ord(a^k) = n / (k, n)   — где (k, n) = gcd(k, n)

Доказательство

        (a^k)^m = e  m ∈ N

                ↕️

        a^{km} = e

                ↕️

        n | km  (d = gcd(k, m))

                ↕️

        n/d | gcd(k/d m)

        Самое маленькое m ∈ N — это n / d

Свойство 8
        ab = ba,  gcd(ord(a), ord(b)) = 1
                
        ord(ab) = ord(a) ord(b)

Подгруппа порожденная подмножеством

        G — группа, S \subset G

Используем:

Теорема
        Если H_α, α ∈ Λ, — подгруппа G, то

                \intersect H_α  —  подгруппа G
                   α ∈ Λ

        Положим

                <S> = \intersect H
                       S \subset H
        
        То есть пересечения по всем подгруппам G, содержащим S.

        <S> — минимальная подгруппа G, содержащая S

Пример
        Z = <N>
        < \emptyset > = {e}

        Пусть S != \emptyset. Тогда

        <S> = { g ∈ G | g = s1^k1 s2^k1 ... sn^kn, n ∈ N, si ∈ S, ki ∈ Z }

Определение
        Пусть G = <S>

        Если S конечно, то G конечно порожденная

Определение
        Группа, порожденная одним элементом, называется циклической

                Если S = {a},

        вместо <S> пишем <A>

Определение
        Группа G циклическая, если ∃a ∈ G: ∀x ∈ G, ∃k ∈ Z: x = a^k
        
        a называется образующим элементом G. Пишут G = <a>

Структура

        Пусть G = <a>

Случай 1
        ord(a) = n ∈ N:

                G = { e, a, ..., a^{n-1} }  (и |G| = n)

Случай 2
        ord(a) = ∞:

                G = { ..., a^{-2}, a^{-1}, e, a, a^2, ... }

Пример 1
        Z_n = <[1]n>

Пример 2
        Z = <1>

Пример 3
        U = < \epsilon >

Теорема
        Пусть |G| < ∞ (G - группа)
        G циклическая
                ↕️
        ∃a ∈ G: ord(a) = |G|

Теорема (об образующих конечной циклической группы)
        Пусть G - конечная циклическая группа

                G = <a>, |G| = n

        Элемент a^k, k ∈ Z — образующий тогда, и только тогда, когда (k, n) = 1

        В G ровно φ(n) образующих элементов.

Доказательство
        1) Пусть a^k образующий  <=>  |a^k| = n  <=>  |a| / (k, |a|) = n  <=> |a| = 1

        Тогда

        n / (k, n) = n <=> (k, n) = 1

        G = {e, a, ..., a^(n-2), a^(n-1)}

        a^k,  0 <= k <= n - 1  <=>  (k, n) = 1

        Таких k ровно φ(n)

        
Теорема (об образующих бесконечных циклических групп)
        Пусть G - бесконечная циклическая группа, G = <a>.

        Образующими элементами G являются a, a^(-1), и больше ничего.
        
Доказательство
        Ясно что G = <a^(-1)>

        Пусть a^k — образующий элемент G. Тогда ∃m ∈ Z: a = (a^k)^m = a^(km)

        Так как |a| = ∞, 1 = km, тогда k | 1 и тогда k = 1 или k = -1

Теорема (о цикличности подгрупп циклической группы)
        Любая подгруппа циклической группы тоже циклическая

Доказательство
        Пусть G = <a>, H — подгруппа G

        Если H = {e} то H = <e>

        Пусть H != {e}
        Возьмем 

        x ∈ H, x != e, ∃m ∈ Z: x = a^m  =>  a^(-m) ∈ H  =>  ∃k ∈ \N: a^k ∈ H
        
        Пусть s — наименьшее натуральное такое, что a^s ∈ H

        Покажем, что H = <a^s>

        Ясно, что <a^s> \subset H

        Обратные вложения:

                Берем x ∈ H, тогда ∃n ∈ \Z такое, что x = a^n

        Поделим n на s:

                n = sq + r, 0 <= r <= s-1

        x = a^(sq + r) = a^(sq) * a^r  =>  a^r = a^(-sq) x = a^s(^-q) x

        a^s ∈ H
        a^s^(-q) ∈ H
        x ∈ H

        Отсюда вытекает, что r = 0, иначе будет противоречие выбору s.

Теорема (о подгруппах бесконечных циклических групп)
        Пусть 🍔 = <a> — бесконечная циклическая группа

        Подгруппы <a^k>, k >= 0 группы 🍔, попарно различны.

        Это все подгруппы 🍔.

Доказательство
        Пусть k,m ∈ \N и <a^k> = <a^m>. Нужно доказать что тогда k = m. Раз они все попарно
        различны.

        a^k ∈ <a^m>  =>  ∃n ∈ \Z:  a^k = a^(mn). Так как |a| = ∞, k = mn  =>  m | k

        Аналогично

        a^m ∈ <a^k>  =>  ∃n ∈ \Z:  a^m = a^(kn). Так как |a| = ∞, m = kn  =>  k | m

        Тогда m = k

        Ясно, что ∀k ∈ \N  <a^k> != <a^0> = {e}

Следствие
        По доказательству предыдущей теоремы

        <a^k>, k ∈ \N — все подгруппы G.


Теорема (о подгруппах конечной циклической группы)
        Пусть G — конечная циклическая группа.

        G = <a>, |G| = n

        (i) Для ∀ k | n (k ∈ \N) ∃! подгруппа H группы G такая, что |H| = k

        При этом H = <a^(n/k)>  и  H = {x ∈ G | x^k = e}

        (ii) Подгруппы как в (i) — это все подгруппы G

# Гомоморфизмы
        
Определение
        Пусть G1, G2 — группы.
        Отображение f: G1 -> G2 называется гомоморфизмом G1 в G2, если
        выполняется равенство

                f(ab) = f(a) f(b)  ∀a,b ∈ G1

        Hom(G1, G2) — множество всех гомоморфизмов G1 в G2

Свойство 1
        f(e1) = f(e2)

Доказательство
        f(e1) = f(e1e1) = f(e1)f(e1) =  (умножим на f(e1)^(-1))

Свойство 2
        f(a^(-1)) = (f(a))^(-1), ∀a ∈ G1

Свойство 3
        Если a ∈ G1, |a| < ∞, то f(a) < ∞ то |f(a)| | |a|  (делит)

Свойство 4
        f ∈ Hom(G1, G2), g ∈ Hom(G2, G3), то gf ∈ Hom(G1, G3) (композиция g и f)

# Ядро гомоморфизма ☀️

Определение
        Для f ∈ Hom(G1, G2) kernel(f) = {x ∈ G1 | f(x) = e2}

Теорема
        kernel(f) — подгруппа G1

Теорема
        Пусть f ∈ Hom(G1, G2). f: G1 -> G2 инъективно <=> kernel(f) = {e}

# Изоморфизмы

Определение
        Биективный f ∈ Hom(G1, G2) называется изоморфизмом G1 на G2.

                                                         
        Если существует изоморфизм G1 на G2, то пишут G1 ~= G2 (но только у меня в конспекте 🤓)

Свойство 1
        Пусть f — изоморфизм G1 на G2, g — изоморфизм G2 на G3, то

                gf — изоморфизм G1 на G3

Свойство 2
        f — изоморфизм G1 на G2  =>  f^(-1) — изоморфизм G2 на G1

Свойство 3
        Пусть f — изоморфизм G1 на G2, тогда

                ∀a ∈ G1 |a| = |f(a)|

Теорема (об изоморфности конечных циклических групп)
        Пусть G — конечная циклическая группа, |G| = n. Тогда G ~= Zn

Доказательство
        Для класса k ∈ Zn положим

                f(k) = a^k  (0 <= k <= n-1)

        Где a — образующий элемент G (G = <a>)

        Все a^k попарно различны, f: Zn -> G биективно. Покажем, что f ∈ Hom

        ∀k,m ∈ Zn, 0 <= k, m <= n - 1

                f(k + m) = a^((k + m) mod n) = (так как |a| = n) = a^(k + m) = a^k * a^m = f(k) f(m) 

        Zn изоморфно G, тогда и G изоморфно Zn

Следствие
        Любые две циклические группы одного конечного порядка изоморфны между собой

# Уравнение x^2 = b


        G = <a>, |G| = m

        Пусть

                G2^+ = { y ∈ G | y^(m/2) = e }

                G2^- = { y ∈ G | y^(m/2) != e }

        (Если 2 | m)

Теорема
        (I) Пусть m нечетное. Тогда уравнение x^2 = b имеет одно решение ∀b ∈ G

        (II) Пусть m четное.
                 i) Уравнение x^2 = b имеет решение в G <=> b ∈ G2+
                    Если b^(m/2) = e, то оно имеет 2 решения. Если x0 — решение,
                    то другим решением является x0 * u, где u — единственный
                    элемент G порядка 2.
                ii) Уравнение x^2 = b не имеет решений в G <=> b^(m/2) = u
               iii) G2^+ — подгруппа G и |G2^+| = m/2

Доказательство
        (I) Пусть m нечетное ==> (m, 2) = 1. А 2 — это n из предыдущей теоремы.
            Это имеет место по предыдущему следствию.
       (II) Пусть m четное ==> d = (m, 2) = 2
                i) По теореме x^2 = b разрешимо <=> b^(d/2) = e, т.е b ∈ G2^+
                Пусть |u| = 2 (такое u ∈ G существует, их φ(2) = 1)

               ii) Имеем: если b^(m / 2) != e, то b^m = e и b^(m/2) != e =>
                => |b^(m/2)| = 2

                Итак, если для b решений нет, то b^(m/2) = u. Наоборот, если b^(m/2) = u,
                то b^(m/2) != e и решений нет.

              Возвращаемся к i): (про вид решений)

                Пусть b^(m/2) = e. Пусть x0 — одно решение.  

                (x0 u)^2 = x0^2 u^2 = x0^2 = b

              ==> x0 u — такое решение. При этом

                x0 != x0 u (ведь u != e)

              Значит x0, x0 u — это все 2 решения.

              iii) Напоминание:

                {x ∈ G | x^k = e} — подгруппа G порядка k (если k | |G|)

                У нас G2+ = {y ∈ G | y^m/2 = e} => 
                G2+ — подгруппа G, |G2^+| = m/2

              (╯°□°)╯ __ ┻━┻

Замечание
        G2^- = {y ∈ G | y^(m/2) = u}




# Сравнение x^2  a (mod p)
        (a, p) = 1. p — простое нечетное

        Сводится к G = Z^*_p = {1, 2, ..., p-1}

        |Z^*_p| = p - 1. Она циклическая.

Теорема
        i) Сравнение x^2 = a (mod p) разрешимо <=> a^(p - 1)/2 = 1 (mod )

        Если a^(p-1)/2 = 1 (mod p), то оно имеет 2 решения по модулю p.
        Если x0 — одно из них, то -x0 — другое.

        ii) Сравнениее x^2 = a (mod m) не имеет решений <=> a^((p-1)/2) = -1 (mod p)

        Без доказательство
